## ⚡ Comandos Rápidos

```bash
# Instalar e rodar
npm install
npm run dev

# Verificar qualidade
npm run type-check
npm run lint

# Build e produção
npm run build
npm start

# Deploy
vercel --prod
```

## 🆘 Problemas Frequentes

| Problema | Solução |
|----------|---------|
| Porta 3000 ocupada | `npm run dev -- -p 3001` |
| Módulos não encontrados | `rm -rf node_modules && npm install` |
| Erros de TypeScript | `npm run type-check` |
| Build falha | `npm run lint:fix` depois `npm run build` |

Para mais, consulte **README_DETALHADO.md → Troubleshooting**
